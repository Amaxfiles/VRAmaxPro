# VRAmax PRO
The ultimate PCVR optimization utility designed for simracers and VR enthusiasts using Meta Quest headsets.

## Overview
Tired of manually configuring the Oculus Debug Tool before every session? VRAmax PRO automates elite performance settings, injecting them instantly to ensure a stutter-free, high-fidelity PCVR experience.

## Features
- **1:1 Performance:** Automatically disables ASW (Asynchronous SpaceWarp) to eliminate ghosting.
- **Visual Clarity:** Forces Link Sharpening and optimizes FOV for razor-sharp visuals.
- **Latency Control:** Locks a high-bandwidth constant video bitrate (up to 800 Mbps).
- **Customization:** Choose between extreme, high, or medium presets, or define your own.

## Installation
1. Download the latest `VRAmaxPro.exe` from the [Releases] page.
2. Run as Administrator to allow registry modifications.
3. Select your Meta Horizon CLI path and click "Inject".
